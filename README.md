# DryPort

DryPort create using program NodeJS framework ExpressJS

## Installation

1. Clone git
```
https://github.com/hendras1722/BackendNodeJS
```

2.Open terminal, type 
```
npm install
```

3. add file .env in ./footballscore 
```
DB_HOST = 'localhost'
DB_USER = 'root'
DB_PASSWORD = ''
DB_NAME = 'dryport'

PORT = 3000
```


## Running
open your terminal. type 
```
npm start
```
